HTML

  • The head element contains informaton about the webpage.
  • The body element represents the visible content shown to the user.
  • Any element nested within another elements is known as the Child element.
  • Alt text is text that is not seen by the reader of a webpage but for a screen reader for disabilities.
  • HREF is for links

CSS

  • There are 3 ways to style a web[age with CSS. Inline CSS, Internal CSS, and External CSS.
  • Inline is for an individual line of text, Internal is for the whole page, and External is the most common because it allows developers to keep all their CSS rules in a seperate file, which makes deisgn changes easier.
  • External CSS needs a link placed in HTML's < head >
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and swithces to it

JavaScript

  • To declare a function, we use the function keyword and then give our function a unique name followed by parentheses. Then we add curly brackets {} that contain the code for the task we want to perform
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
